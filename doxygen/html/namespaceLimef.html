enum class Limef::HWAccel

Hardware acceleration backends.

Note: This is for FFmpeg-based hardware acceleration only. V4L2 backend is separate (use V4L2EncoderParams/V4L2DecoderParams instead).

Create hardware device context.

Creates an AVBufferRef* for the specified hardware accelerator. Caller owns the returned context and must free it with av_buffer_unref().

◆ getAvailableHWAccel()

std::vector< HWAccel > Limef::getAvailableHWAccel ( )

Get available hardware accelerators.

Checks both compile-time support and runtime availability:

  • Device exists and can be opened
  • Has encoding/decoding capability

Results are cached after first call.

Returns
std::vector<HWAccel> List of available accelerators (always includes SW)
