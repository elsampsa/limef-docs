Detailed Description

template<typename T>
class Limef::CircularBuffer< T >

An efficient circular buffer implementation.

CircularBuffer is more efficient than std::queue implementations. Based on std::vector. Max number of elements needs to be known in advance. -> push, pop and front operations must always be checked in advance
