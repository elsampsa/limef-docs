Detailed Description

Bidirectional pipe for interprocess/interthread communication using JSON.

Creates a bidirectional pipe similar to Python's multiprocessing.Pipe. Used for communication between threads or processes using JSON strings. Thread-safe.

Example usage:

auto [pipe1, pipe2] = Pipe::create();
// In thread 1
pipe1.send("{\"command\": \"start\", \"value\": 42}");
// In thread 2
std::string json_message;
pipe2.recv(json_message); // json_message = "{\"command\": \"start\", \"value\": 42}"
// Then parse json_message with your favorite JSON library
