Detailed Description

FrameFifo specialization for DecodedFrame with GPU buffer pool support.

Author
Sampsa Riikonen
Date
2024
Version
1.6.1

When GPU-decoded frames flow through FrameFifo, this class ensures that stack frames own their own GPU buffers via a dedicated hw_frames_ctx pool. This prevents data corruption when the upstream decoder reuses its buffers.
