Detailed Description

V4L2 Memory-to-Memory Decoder implementation.

Author
Sampsa Riikonen
Date
2025
Version
1.6.1

Uses the stateful V4L2 M2M API for hardware decoding. Works with:

  • vicodec (virtual codec for testing, uses FWHT)
  • Jetson hardware decoders (H.264, H.265)
  • Other V4L2 M2M capable devices

NOTE: This decoder operates independently from FFmpeg. It does NOT require AVStream - codec info comes from V4L2DecoderParams. Typical use case: decode packets from V4L2Encoder.
