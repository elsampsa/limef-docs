Detailed Description

Configuration for X11 presenter threads (EGLPresenterThread, GLXPresenterThread).

Collects all window and timing parameters in one place so the thread constructors stay concise as new options are added.

Example:

PresenterContext pctx;
pctx.width = 1920;
pctx.height = 1080;
pctx.bypass_compositor = true; // needed on PRIME/KWin for GLX
GLXPresenterThread glx("glx", FrameFifoContext(false, 20, 40), pctx);
