Detailed Description

V4L2 Memory-to-Memory Encoder implementation.

Author
Sampsa Riikonen
Date
2025
Version
1.6.1

Uses the stateful V4L2 M2M API for hardware encoding. Works with:

  • vicodec (virtual codec for testing, uses FWHT)
  • Jetson hardware encoders (H.264, H.265)
  • Other V4L2 M2M capable devices
