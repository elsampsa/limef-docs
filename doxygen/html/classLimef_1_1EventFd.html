Detailed Description

File descriptor wrapper for poll/select multiplexing.

Wraps Linux eventfd for signaling between threads or for waking up poll/select loops. Useful for multiplexing FrameFifo reads with socket I/O.

Usage:

EventFd event;
// In producer thread:
event.set(); // Signal data available
// In consumer thread with poll:
struct pollfd fds[2];
fds[0] = {socket_fd, POLLIN, 0};
fds[1] = {event.getFd(), POLLIN, 0};
poll(fds, 2, timeout);
if (fds[1].revents & POLLIN) {
event.clear();
// Process fifo data
}
