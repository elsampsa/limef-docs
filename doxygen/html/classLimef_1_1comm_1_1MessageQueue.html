Detailed Description

template<typename MessageType>
class Limef::comm::MessageQueue< MessageType >

Thread-safe message queue using pipes.

Provides a higher-level interface for sending messages between components with support for message handlers. Can be used for communication between filters, threads, or Python integration.

Example usage:

// Create a message queue
MessageQueue<std::string> queue;
// Set up a handler
queue.setHandler([](const std::string& msg) {
std::cout << "Received: " << msg << std::endl;
});
// Start the queue
queue.start();
// Send messages from anywhere
queue.send("Hello from framefilter!");
queue.send("Hello from thread!");
// Later, stop the queue
queue.stop();
