Detailed Description

template<typename... Args>
class Limef::comm::Caller< Args >

Qt-style signal/callback mechanism for event notification.

Allows multiple callbacks to be registered and invoked when an event occurs. Thread-safe for registration and emission.

Example usage:

// Declare callers in your class
Caller<int, std::string> streamRequested; // slot, tail
Caller<int> lastClientLeft; // slot
// Connect callbacks (from main thread, before starting)
server.streamRequested.connect([&](int slot, const std::string& tail) {
if (slot == 1) mediafile.playStream();
});
// Emit from server thread (invokes all connected callbacks)
streamRequested.emit(1, "/live/camera1");
