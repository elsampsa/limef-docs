Detailed Description

Decoder factory for creating backend-specific decoders.

Author
Sampsa Riikonen
Date
2025
Version
1.6.1

Provides a variant-based API for creating decoders from different backends (FFmpeg, V4L2) using a type-safe interface that works well with Python bindings.

All param types are always available in the variant - this keeps the API stable and avoids ifdef permutation explosion. Runtime checks determine if a backend is actually available.
