Detailed Description

A base class for simple one-to-one framefilters.

A FrameFilter of this class can be connected to only one framefilter in the framefilter chain, i.e.

FrameFilter ff1();
FrameFilter ff2();
FrameFilter ff3();
FrameFilter ff4();
ff1.cc(ff2).cc(ff3).cc(ff4)
