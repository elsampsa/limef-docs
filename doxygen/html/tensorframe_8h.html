Detailed Description

TensorFrame — general-purpose multi-plane tensor frame, decoupled from FFmpeg.

Author
Sampsa Riikonen
Date
2025
Version
1.6.1

Overview

TensorFrame holds N independent planes of arbitrary shape and dtype. Each plane lives entirely on CPU (data_) or entirely on GPU (d_data_). All planes in one TensorFrame reside on the same device.

One TensorFrame represents one media stream (video OR audio — not both):

  • Video: typically one plane, shape (C, H, W), dtype Float32 or UInt8
  • Audio: typically one plane, shape (samples, channels), dtype Float32

Memory management

Buffers are grow-only: reused when the new content fits, reallocated only when larger. Contiguous (C-order) strides are computed automatically by reserveCPUPlane() / reserveGPUPlane().

C++ interop

Plane fields are public. Consumers read them directly:

const auto& p = tf.planes[0];
cv::cuda::GpuMat mat(p.shape[0], p.shape[1], CV_8UC1, p.d_data_, p.strides[0]);
